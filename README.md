# gohoro
For guys who crazy with horoscope. Now, you can check your daily horosope every second, just a command.

# Usage
Get :

`go get github.com/nguyenvanduocit/gohorocli`

Run

`gohorocli`

Or

`gohorocli --sign=leo`
